## 1Ô∏è‚É£ Predlog `CREATE TABLE` (prazna tabela sa punim modelom)

```sql
  id              serial4       PRIMARY KEY,
  direction       text          NOT NULL,   -- 'DE_SR' ili 'SR_DE'
  headword        text          NOT NULL,   -- lema (osnovni oblik)
  main_gloss      text          NULL,       -- prvo osnovno znacenje
  plain_text      text          NOT NULL,   -- rezime svih znacenja (srpski)
  content_html    text          NOT NULL,   -- HTML za prikaz
  image_url       text          NULL,       -- opcionalno

  is_irregular    boolean      DEFAULT false,
  has_audio       boolean      DEFAULT false,
  frequency_rank  integer,
  created_at      timestamptz  DEFAULT now(),
  updated_at      timestamptz  DEFAULT now()
);
```
